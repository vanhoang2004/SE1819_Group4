Question Template file: Download Excel File

Instruction

1. Tải File Template

Để bắt đầu, bạn cần tải file template questionTemplate.xlsx. Đây là file mẫu chứa cấu trúc dữ liệu chuẩn để nhập các câu hỏi.

  • Bấm vào liên kết "Download Excel File".

2. Mở và Chỉnh Sửa File Template

Sau khi tải file questionTemplate.xlsx, bạn cần mở file này bằng phần mềm Excel hoặc bất kỳ chương trình nào hỗ trợ định dạng file .xlsx.

Các bước chỉnh sửa:

  • Mở file questionTemplate.xlsx.
  • Bạn sẽ thấy một bảng với các cột tiêu đề như "Question Title", "Image", "Option 1", "Option 2", "Option 3", "Option 4", "Answer" và "Level".
  • Nhập các thông tin câu hỏi vào các dòng bên dưới tiêu đề. Mỗi dòng sẽ tương ứng với một câu hỏi.
    • Question Title: Nội dung của câu hỏi.
    • Option 1,2,3,4: Các lựa chọn trả lời cho câu hỏi.
    • Answer: Câu trả lời đúng.
    • Level: Chọn 1 trong 3 level: Nhận biết, Vận dụng và Thông hiểu

3. Import File Vào Hệ Thống

Bạn cần import file questionTemplate.xlsx đã chỉnh sửa lên hệ thống

Các bước import file:

  • Bấm vào nút Choose File và chọn file template đã chỉnh sửa từ máy tính của bạn.

Lưu ý:

  • Đảm bảo rằng file của bạn tuân thủ đúng cấu trúc template: Mọi thay đổi khác với template hệ thống sẽ không ghi nhận câu hỏi .

Xử Lý Lỗi Thường Gặp

  • File không đúng định dạng: Đảm bảo rằng bạn đã lưu file với định dạng .xlsx.
